# 🌦️ Weather Dashboard

A modern, responsive Weather Dashboard built using **HTML, CSS, and JavaScript**.  
This project uses the **OpenWeather REST API** to fetch real-time weather data with asynchronous JavaScript (async/await).

---

## 📌 Features

- 🌍 Search weather by city name
- 📍 Auto-detect user location (Geolocation API)
- 🌡️ Real-time weather data (temperature, humidity, wind speed)
- 📊 5-day weather forecast
- 🎨 Dynamic weather icons from API
- 🌙 Dark / Light mode toggle
- ⚡ Fully responsive design (mobile, tablet, desktop)
- 🛡️ Error handling for invalid cities and network issues

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid + Variables)
- JavaScript (ES6+)
- OpenWeather API

---

## ⚙️ How It Works

1. User enters a city name or uses location button  
2. App sends request to OpenWeather API  
3. Data is fetched using `fetch()` + `async/await`  
4. JSON response is parsed and displayed dynamically  
5. Forecast data is processed and rendered in UI  

---

## 🌐 API Used

- OpenWeather Current Weather API  
- OpenWeather 5-Day Forecast API  

🔗 https://openweathermap.org/api
